"This work marks the transition of the project from Awakened Rotation Cosmology (ARC) to Lagrangian Variable Cosmology (LVC). While ARC introduced a dynamical phase variable framework to address the structural limitations of fixed expansion models, LVC advances this approach into a two-channel formulation that consistently connects CMB microphysics, BAO observables, and supernova distances. By unifying geometric and fluid sectors through a single rotational variable and an invariant coupling relation, LVC establishes a more complete and observationally viable framework for resolving the CMB–BAO tension."
